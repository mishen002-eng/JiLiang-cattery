# Jiliang Cattery — Design System

## Color Palette

| Token | Hex | Usage |
|---|---|---|
| `brand-cream` | `#FAF7F2` | Page background |
| `brand-charcoal` | `#2C2C2C` | Primary text, headings |
| `brand-slate` | `#6B7B8D` | Secondary text, muted content |
| `brand-slate-light` | `#8A9BAD` | Tertiary text, placeholders |
| `brand-brass` | `#B8956A` | Primary accent, CTAs, links |
| `brand-brass-light` | `#D4B896` | Hover states, subtle accents |
| `brand-brass-dark` | `#9A7B55` | Active/pressed states |
| `brand-ice` | `#E8EDF2` | Card backgrounds, light panels |
| `brand-ice-dark` | `#D1D9E2` | Borders, dividers |
| `brand-white` | `#FFFFFF` | Card surfaces, overlays |

## Typography

- **Headings:** Playfair Display (serif) — bold, tight tracking
- **Body:** Inter (sans-serif) — 16px base, 1.6 line height
- **Scale:** h1 (4xl–6xl), h2 (3xl–4xl), h3 (2xl–3xl), h4 (xl–2xl)

## Spacing

- `section-padding`: py-16 md:py-24 with px-4 md:px-6
- `container-narrow`: max-w-5xl centered
- `container-wide`: max-w-7xl centered

## Radius

Base radius: 0.625rem (10px). Scaled via shadcn's radius system.

## Breakpoints

Standard Tailwind: sm (640), md (768), lg (1024), xl (1280), 2xl (1536).
